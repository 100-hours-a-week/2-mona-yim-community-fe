## 🛠️ **설치 및 실행 방법**

1. 📥 **레포지토리 클론**

    ```bash
    git clone https://github.com/100-hours-a-week/2-mona-yim-community-fe.git
    cd 2-mona-yim-community-fe

    ```

2. 🏗️ **필요한 의존성 설치**

    ```bash
    npm install

    ```

3. 🚀 **앱 실행**
    ```bash
    npm start
    ```
